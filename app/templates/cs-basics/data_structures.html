  • Array
    • Array is collection of items stored at contiguous memory locations
    • Advantages:
      ○ Allows random access of elements. This makes accessing elements by position faster
      ○ Have better cache locality , that makes pretty big difference in performance

      Syntax in C/C++ : refer
      ○ declatation by size : int arr[10];
      ○ declaration by initializing elements : int arr = {1,2,3,4};
      ○ declaration by specifying size and initializing elements : int arr[6] = {1,2,3,4}

      Syntax in python: refer
      ○ In python array can be handled using the module "array"
      ○ import array ○ arr = array.array('i', [1, 2, 3])
      ○ append() -> used to add the value mentioned in the arguments at the end of array
      ○ insert(i,x )->adds the value x in the array at specified position i
      ○ pop() -> removes the element at the position mentioned in the arguments
      ○ remove() -> removes the first occurance of value mentioned in argument
