    Object-based vs Object Oriented: Object-based programming languages follow all the oops except the inheritance. Ex: Javascript, VBScript
  • Object: Real world entity such as pen, chair, table, computer, watch etc.,
  • Object-Oriented Programming: Methodology or paradigm to design a program using classes and objects. Simplifies software development and maintanence by providing some concepts
      • Objects and Classes:
    • Object: It is both physical and Logical Entity. Entity which has state and behavior is object.
      Object has three characteristics - state, behavior, identity
      Object is an instance of class
      Class in java can contain:
      Fields
      Methods
      Constructors
      Blocks
      Nested class and interface
    • Class: It is only Logical Entiry. Class is a template or blue print from which objects are created. Group of objects which has common properties
    • Inheritance: When one object aquires properties and behaviors of a parent object. Provides code reusability
    • Polymorphism: If one task is performed in different ways, it is known as Polymorphism. Method overloading and method overriding in java are used to acheive Polymorphism
    • Abstraction: Hiding internal details and showing functionality is known as Abstraction. In java abstract classes and interfaces are used to acheive Abstraction
    • Encapsulation: Binding or wrapping code into a single unit is called as encapsulation.
