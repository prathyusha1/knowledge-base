  • Interpreter or compiler is a program that converts high level program in to machine code understood by computer
  • Interpreter -
    ○ Translates the program one statement at a time
    ○ No intermediate object code is generated, hence it is memory efficient
    ○ Continues translating the program until first error is met
  • Compiler :
    ○ Scans the entire program and translates it as a whole into machine code
    ○ Generates intermediate object code, requires more memory
    ○ Generates error messages only after scanning the whole program
