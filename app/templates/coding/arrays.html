  1. Array Rotation:
    ○ Solution :
    • In-place rotation (refer) - in this solution the array is rotated in-place by computing the new index value for the current index
      ○ New index is calculated by using the formula
      ○ newIndex = (currentIndex - (no_of_rotation % array_length) + array_length) % array_length
      ○ currIndex is looped through the length of array, and swap the elements of (currentIndex,newIndex)
      ○ Time Complexity : O(n)
      ○ Space Complexity : O(1)
    • Array Reversal Algorithm (refer) - in this solution array is divided in to two parts
      ○ Input Arr - array, d- number of rotations
      ○ A = [0,d-1] B=[d,n-1]
      ○ Reverse array A => Arev
      ○ Reverse array B => Brev
      ○ Combine Arev,Brev => Ares = [Arev,Brev]
      ○ Reverse Ares, the result of this step will be the output array
      ○ Time Complexity : O(n)
