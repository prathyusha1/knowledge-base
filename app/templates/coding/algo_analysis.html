Ways to find out the betterness of an algorithm compared to other one
  1. Asmptotic Analysis : Behavior of algorithm as it approaches infinity
    In this approach of analysis, the performance of an algorithm is evaluated in terms of input size.i.e., calculate time(or space) taken by an algo increase with input size
    Example : How to compare linear search and Binary search of a sorted array with asymptotic analysis
    Run linear search on fast computer and binary search on slow computer, for small values of input array size n fast computer may taken less time but after certain value of input size binary search will take less time comparatively, since the machine dependent constants can be ignored after certain value of input
  2. Analysis based on worst case, best case and average case
    Worst case analysis : Calculate the upper bound on running time of algorithm, i.e., identify the case that causes maximum number of operations
    Average Case Analysis: Take all possible inputs and calculate computing time for all the inputs (sum all calculated values)/(total number of inputs)
    Best case Analysis: Calculate lower bound on running time of algorithm
